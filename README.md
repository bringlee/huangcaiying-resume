# 黄才莹个人简历网站

一个现代化、响应式的个人简历网站，展示会计/财务专业背景与工作经验。

## 在线预览

🌐 **Live Site:** https://bringlee.github.io/huangcaiying-resume/

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。

## 技术栈

- HTML5 语义化标签
- CSS3 现代特性（Flexbox / Grid / 渐变 / 动画 / 毛玻璃效果）
- Vanilla JavaScript（滚动动画、数字计数、响应式菜单）

## 部署

通过 GitHub Actions 自动部署到 GitHub Pages。
